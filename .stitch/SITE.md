# ClinedOps Site Plan

## 1. Vision
An authoritative, clinician-first ops center for managing teaching hours, surveys, and reporting. The design moves away from utility grids into an editorial storytelling experience for medical educators.

## 2. Project Context
- **Stitch Project ID**: `2581763650719485882`
- **Framework**: Vite + React + Tailwind + Shadcn
- **API**: FastAPI Backend (running at `localhost:8000`)

## 3. Sitemap (Design Source)
- [ ] **Login Screen**: Authoritative clinician entry point (`login-screen.html`)
- [x] **Overview / Tutor Dashboard**: Home page for clinicians (`index.html`)
- [ ] **Teaching Hours Logging**: Form for logging single/bulk sessions (`teaching-hours.html`)
- [ ] **Reporting & Analytics**: Data-intensive center for admins (`analytics.html`)
- [ ] **Student Portal**: Attendance verification for students (`student-portal.html`)
- [ ] **Programme Admin Center**: Sarah's overview dashboard (`admin.html`)

## 4. Component Inventory (Roadmap)
- [ ] **Navigation Shell**: Responsive layout with Glass header and fixed/collapsible sidebar.
- [ ] **Impact Cards**: Metric cards with progress bars and tonal layering.
- [ ] **Clinical Ledger**: Borderless tables with high legibility.
- [ ] **Form Elements**: Custom shadcn inputs with `surface-highest` background.
